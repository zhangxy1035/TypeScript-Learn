什么是 TypeScript？

TypeScript 是一种由微软开发的自由和开源的编程语言，它是JavaScript的一个超集，扩展了JavaScript的语法。

语法特性

  • 类 Classes
  • 接口 Interfaces
  • 模块 Modules 
  • 类型注解 Type annotations
  • 编译时类型检查 Compile time type checking 
  • Arrow 函数 (类似 C# 的 Lambda 表达式)

动手实施

当自己编写一个TypeScript文件时，需要将自己的文件进行编译。

例如:编译文件为hello.ts,就会在与hello.ts同级目录下出现hello.ts文件。

命令: tsc hello.ts
